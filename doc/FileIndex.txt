NAME

        use Beyonwiz::Recording::FileIndex;

SYNOPSIS

    Provides access to the Beyonwiz recording file index via HTTP.

SUPERCLASS

    Inherits from "Beyonwiz::Recording::Index"

METHODS

    "Beyonwiz::Recording::FileIndex->new($base)"
        Create a new Beyonwiz recording index object. $base is the base URL
        for the Beyonwiz device.

    "$i->path([$val]);"
        Returns (sets) the recording path.

    "$i->load;"
        Load the index from the Beyonwiz.

PREREQUISITES

    Uses packages: "Beyonwiz::Recording::Header",
    "Beyonwiz::Recording::Trunc", "Beyonwiz::Recording::Index",
    "Beyonwiz::Recording::IndexEntry", "Beyonwiz::Utils", "LWP::Simple",
    "URI", "URI::Escape", "File::Spec::Functions".

BUGS

    Uses a fixed value for the path name of the index, rather than deriving
    it from *locationURL* in "Beyonwiz::WizPnP".
